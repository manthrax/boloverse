Info: ryan2autopilot.2.4.3.windows.i386.zip or ryan2autopilot2.4.3.linux.i386.tar.gz
	Ryan2 is an autopilot for the game Win/LinBolo.  It can run on either
	Windows in versions 9/10 of the game on in linux on version 9 (and most
	likely 10 if it ever comes out.)  To install ryan2, simple copy ryan2.brn
	or ryan2.so (for windows or linux respectively) to your bolo/brains
	directory, and start bolo.  Then go to the Brains menu and select Ryan2.
	If you have any questions, feel free to e-mail me at rstout@cybernet1.com
	To check for newer versions, check out Canucks Unofficial WinBolo FTP at
	ftp://winbolo.dhs.org/
